Tipos de Links

Links Externos

Link Externo aponta para outro Site, que não é seu.

Exemplos:
1- Acessando o repositório público no GitHub do Gustavo Guanabara.
Esse Site será sobreposto ao Site Original, onde está o Link.

2- Acessando o Canal no YouTube do Gustavo Guanabara.
Esse Site não será sobreposto, pois será aberto em uma nova página.

Links Internos

Link Interno relaciona outra página que está dentro do nosso servidor e diz respeito à nossa página.

Este é a primeira página do site. Se você quiser, pode acessar a segunda página.

Acesse a página de notícias.
