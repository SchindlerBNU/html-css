Principais Formatações

Elemento B - Negrito

Nesta frase, temos um termo em negrito usando o Elemento <b>.
O Elemento <b> é uma TAG NÃO SEMÂNTICA, ou seja, é uma FORMA.

Elemento Strong - Destaque

Nesta frase, tenho um termo em destaque usando o Elemento <strong>.
O Elemento <strong> é uma TAG SEMÂNTICA.

Elemento I - Itálico

Nesta frase, temos um termo em itálico usando o Elemento <I>.
O Elemento <I> é uma TAG NÃO SEMÂNTICA.

Elemento EM - Ênfase

Nesta frase, temos um termo em enfase usando o Elemento <EM>.
O Elemento <EM> é uma TAG SEMÂNTICA.

Elemento MARK - Marcar Textos

Podemos criar um Texto Marcado usando o Elemento <MARK>. O Elemento <MARK> é uma TAG SEMÂNTICA.

E no outro texto, temos um Texto Marcado, de cor diferente ao do padrão

E em outro texto, temos um Texto Marcado, da cor padrão do Elemento <MARK>, amarelo.

Texto Grande e Texto Pequeno

Estamos criando um Texto Grande e um Texto Pequeno neste parágrafo.

Texto Deletado

Podemos marcar um Texto como Deletado/Excluído para indicar que ele deve ser lido, mas não considerado.

Texto Inserido

Podemos marcar um Texto como Inserido para dar uma ênfase e indicar que ele foi adicionado depois.
O Elemento <INS> é uma TAG SEMÂNTICA.

Existe, também, o Sublinhado com o Elemento <U>, NÃO SEMÂNTICO.
