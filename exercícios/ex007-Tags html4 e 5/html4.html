Teste de Marquee

Atributo <BGCOLOR>: Funciona, mas está Descontinuado.
Elemento <CENTER>: Funciona, mas está Descontinuado.
Elemento <FONT>: Funciona, mas está Descontinuado.
Propriedade COLOR: Funciona, mas está Descontinuado.
Elemento <U>: Funciona, mas está Descontinuado e substituído pelo <ADRESS>.
Elemento <ARTICLE>: Funciona e ainda é utilizado no HTML5.
Elemento <ADRESS>: Funciona e ainda é utilizado no HTML5, substituindo o Elemento <U>.
Elemento <MARQUEE>: Funciona, mas está Descontinuado, devendo ser substituído pelo CSS MARQUEE STYLE
Exemplo:
<DIV class="marquee"><SPAN>Lorem Ipsum</SPAN></DIV>.
No CSS se marca:
.marquee span{display: inline-block;
padding-left: 100%;
animation: marquee 15s linear infinite;}
@keyframes marquee{
0% {
transform: translate(0, 0);
} 100%{
transform: translate(-100%, 0);
}
