Outras Formatações

Elemento <CODE> - Fonte Monoespaçada

Código-Fonte é o conjunto de palavras ou símbolos escritos de forma ordenada, contendo instruções em uma das Linguagens de Programação existentes, de maneira lógica.
Exemplo:
O Comando document.getElementById('teste') é um Código-Fonte escrito em JavaScript.

Escrito com Fonte Monoespaçada, document.getElementById('teste'), todas as letras possuem o mesmo espaçamento.

Elemento <PRE>

A Tag <pre> é utilizada para representar texto pré-formatado.
O texto é exibido da mesma maneira em que o texto original foi disposto no arquivo ou no Editor de Código Fonte.

Exemplo: 

                
                num = int(input('Digite um número'))
                if num % 2 == 0
                    print(f'O número {num} é PAR')
                else:
                    print(f'O número {num} é IMPAR')
                print('fim do programa')

Elemento <Q> - Citações

A Tag <Q> indica que o texto é uma pequena citação. Coloca o texto entre aspas.

Exemplo:
Como diria o pai de um amigo: o computador é um burro muito rápido.

Elemento <BLOCKQUOTE>Citações Completas ou Longas

A Tag <BLOQKQUOTE> indica que o texto incluído é uma Citação Longa.
Usada em citações de textos de livros, vídeos, fotos e outros. Nota-se um RECUO do texto envolto pela Tag <blockquote>.

Exemplo: Segundo Jeff Noble, no seu livro HTML para leigos:

A diferença entre elementos inline e um bloco de texto é importante.
Os elementos HTML nesse capítulo descrevem os Blocos de Texto.

Pode-se adicionar ao <blockquote> o Parâmetro CITE, que também é uma TAG, serve para definir
uma referência, principalmente, para adicionar a URL de onde foi extraído o texto referido, ou seja, indica a fonte.

O Parâmetro CITE somente será visualizado pelos mecanismos de busca do navegador. Não aparecerá nada e, não será visível no site.

Elemento <ABBR> - Abreviações

Representa uma Abreviação e, opcionalmente, fornece uma descrição completa para a abreviação.

Exemplo:
Estou estudando HTML e CSS. Estou amando!

Note que só será visível a descrição da abreviação ao se passar o mouse por cima.

Elemento <BDO> - Texto Invertido

Bidirectional Override é usado para substituir a direcionalidade atual do texto.
RTL = Right to Left ou LTR - Left to Right.

Exemplo com RTL:
Estou aprendendo a criar coisas em HTML.
