ORDER e ALIGN-SELF

Propriedade ALIGN-SELF
Alinha Itens-Flex (Elementos Filhos) sobreescrevendo o valor Align-Items. Se alguma das Margens do Item-Flex (Elemento Filho) estiver configurado com
Valor = Auto, o Align-Self é Ignorado.

ALIGN-SELF:
Se aplica a cada um dos ITEMS (Elementos Filhos).
Funciona diretamente no Cross-Axis (Eixo-Transversal).
Só é aplicável no Elemento-Filho.

Valores do Align-Self:
*AUTO - Herda a característica de Alinhamento Vertical (Cross-Axis) do PAI. *FLEX-START - Alinhamento no Cross-Start. *FLEX-END - Alinhamento no Cross-End. *CENTER - Alinahdo ao Centro do ELemento-Pai. *STRETCH - Espicha o ITEM para que ocupe todo o espaço do Cross-Axis

Ao se criar Containers Flexbox, a parte externa (Elemento PAI) é FLEX, e a parte interna (Elemento Filho) NÃO é FLEX. Porém, é possível configurar um ou vários Elementos Filhos
e os tornar também Container. Dessa forma, teremos uma Caixa Flexível PAI e em seu interior teremos Elementos-Flexíveis

Propriedade ORDER
>Todo Elemento Filho (ITEM) possui ORDER com Valor: 0 (PADRÃO)
>Ao se colocar valores diferentes em cada Elemento Filho existente dentro do Elemento PAI, como:
Order: 3, Order: 5, Order: -2, Order: 0,
o CSS irá reorganizá-los em ordem crescente.
Assim, os ELementos Filhos ficarão: -2, 0, 3 e 5.

Propriedade Flex: Auto
É aplicada aos Elementos Filhos
