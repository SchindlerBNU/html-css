ALIGN-ITEMS

O Alinhamento dos Elementos Filhos pelo Align-Items
é baseado pelo CROSS-AXIS (Eixo Transversal)

VALORES DO ALIGN-ITEMS: *Stretch (Padrão) = os items são esticados no Eixo Transversal (CROSS-AXIS) *Flex-Start = os ITEMS são posicionados no CROSS-START *Flex-End = os ITEMS são posicionados no CROSS-END *Center = os ITEMS são posicionados no centro do Eixo Transversal

Para entender o Align-Items, deve-se aumentar a alturado container (Elemento PAI)
